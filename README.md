# zaphnath

a simple electron app that is a Bible

This app contains Bible KJV text taken from https://github.com/aruljohn/Bible-kjv

## Usage

1. Clone the repo
2. Run `npm install`
3. Run `npm start` to start the app
4. Select a book then chapter to display verses

## Implementation

- Uses Electron for cross-platform desktop app 
- Fetches JSON representations of Bible books/chapters
- Renders verses with basic navigation
